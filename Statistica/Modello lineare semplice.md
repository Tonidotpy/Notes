# Modello lineare semplice
$$Y|X_{1} = x_{1}, ..., X_{p} = x_{p} \sim N(a_{0} + a_{1}x_{1} + ... + a_{p}x_{p}, \sigma^{2})$$
$$Y|X_{1} = x_{1}, ..., X_{p} = x_{p} \sim N(g(x_{1}, ...,x_{p}; W), \sigma^{2})$$
