# Numeri naturali in base arbitraria
Sia $b \in \mathbb{N}$ diciamo che un [[Rappresentazione dei numeri naturali|numero naturale]] $n \in \mathbb{N}$ si pu√≤ **rappresentare in base $b$** se $\exists k \in \mathbb{N}$ e $\epsilon_{0}, \epsilon_{1}, ..., \epsilon_{k} \in I_{b}= \{ 0, 1, ..., b-1 \}$ se $b \ge 1$ (altrimenti $I_{0} = p$) tale che $n = \epsilon_{0} b^{0}+ \epsilon_{1} b^{1} + ... + \epsilon_{k} b^{k}$ ossia:
$$n = \sum_{i = 0}^{k} \epsilon_{k} b^{k}$$
