# Variabile aleatoria  Bernoulliana
$$P_{X(x)} = \begin{cases} p\ se\ x=1\\ 1 - p\ se\ x = 0\\ 0\ altrimenti \end{cases}$$
$$F_{X(x)} = \begin{cases} 0\ \ \ \ x < 0\\ 1-p\ \ \ \ 0 \le x < 1\\ 1\ \ \ \ x \ge 1 \end{cases}$$
