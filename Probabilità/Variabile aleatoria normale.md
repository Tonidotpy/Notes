# Variabile aleatoria normale
$$f_{X}(x) = \frac{1}{\sqrt{2 \pi \sigma^{2}}} \exp( -\frac{1}{2} \frac{(x - \mu)^{2}}{\sigma^{2}} )\ \ \ \ \mu \in \mathbb{R}\ \ \ \ \sigma^{2} \in \mathbb{R}^{+}$$
