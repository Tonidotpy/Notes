# Variabile aleatoria uniforme
Sia $a < b$:
$$f_{X}(x) = \begin{cases} \frac{1}{b - a}\ \ \ \ se\ x \in [a, b]\\ 0\ \ \ \ altrimenti \end{cases}$$
$$F_{X}(x) = \int_{-\infty}^{x} \frac{1}{b - a} dt$$
