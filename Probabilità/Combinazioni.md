# Combinazioni
- contengono oggetti diversi;
- l'ordine **non** è importante;
- un elemento non può comparire più di una volta.

$$C_{n,k} = \binom{n}{k} = \frac{D_{n,k}}{P_{n}} = \frac{n!}{k!(n - k)!}$$

### Combinazioni con ripetizione

$$C_{n,k}^{r} = \frac{(n + k - 1)!}{k!(n - 1)!}$$
