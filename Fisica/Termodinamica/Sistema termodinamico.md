I sistemi termodinamici vengono classificati in tre categorie:
1. [[Sistema aperto]]
2. [[Sistema chiuso]]
3. [[Sistema isolato]]

Di seguito viene rappresentato lo scambio di materia ed energia per ogni sistema tramite una tabella:

| Sistema/Scambio | Materia | Energia |
| --------------- | ------- | ------- |
| Aperto          | Si      | Si      |
| Chiuso          | No      | Si      |
| Isolato         | No      | No      |
| **Impossibile** | Si      | No      |
